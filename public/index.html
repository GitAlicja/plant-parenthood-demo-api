Plants Demo API

This API was built in order to replace the Trefle botanical API–which has been discontinuedused–used initially for the Plant Parenthood app. Demo API was created for presentational reasons only and allows searching limited number of plants.

Plants you can currently search: lavender, peony, mallow, chamomile and lilac.

Available endpoints are the following:

Method Endpoint Action
GET /api/v1/plants/search Get all the plants
GET /api/v1/plants/search?q={query}&page={number} Get plants whose common name, scientific name or synonym contains the search term
GET /api/v1/plants/:slug Get a single/specific plant
