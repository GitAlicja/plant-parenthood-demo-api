Plants Demo API

Welcome to the Plants Demo API which contains data on selected plants.

The available endpoints are the following:

Method Endpoint Action
GET /api/v1/plants/search Get all the plants
GET /api/v1/plants/search?q={query}&page={number} Get plants whose common name, scientific name or synonym contains the search term
GET /api/v1/plants/:slug Get a single/specific plant
